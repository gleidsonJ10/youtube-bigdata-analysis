# YouTube Big Data Analysis with PySpark

Projeto de análise de dados do YouTube utilizando PySpark e técnicas de Big Data.

## Tecnologias
- PySpark
- Spark MLlib
- PCA
- Regressão Linear
- Parquet

## Funcionalidades
- Tratamento de dados
- Engenharia de atributos
- Machine Learning
- Agregações
- Processamento distribuído
